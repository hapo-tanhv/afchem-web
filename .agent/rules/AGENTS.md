# AFCHEM SCADA Web Application

> Project-specific context for AI agents (Antigravity, Claude Code, etc.). See `.agent/` for workflows and skills.

---

## Project Overview
AFCHEM SCADA System is an ASP.NET MVC 5 portal designed for solar process monitoring, database alarm logging, event logging, and role-based security settings.

---

## Tech Stack
| Layer | Technology |
|-------|------------|
| **Framework** | ASP.NET MVC 5 (.NET Framework 4.7.2) |
| **Language** | C# 6.0 |
| **Database** | MySQL Server 5.6+ |
| **Styling** | Vanilla CSS + Bootstrap 3.4.1 (AdminLTE Master) |
